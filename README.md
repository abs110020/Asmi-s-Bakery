# Asmi-s-Bakery
Bakery landing page using MidJourney, Figma and GSAP scroll animation

- Created images using MidJourney.
- Used Photoshop to add the shadows and save as individual images.
- Used Figma to layout the images and export json file.
- Used GSAP to make the scroll animation
